# Introduction

Grealtly inspired by small multiples (opens new window) from the almighty data visualization world. Small multiple "is a series of similiar graphs or charts using the same scale and axes, allowing them to be easily compared".

One first approach on utilizing this plugin would would be to give the user a bird's-eye view of what one's willing to monitor. Allowing the user to quickly grasp changes and insights.
